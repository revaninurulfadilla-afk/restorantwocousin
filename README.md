# 🍽️ Restaurant POS System - Two Cousin

Sistem **POS (Point of Sales) Restoran** berbasis **PHP & MySQL** untuk membantu proses pemesanan, pembayaran, manajemen menu, stok, dan laporan secara terintegrasi.

## FITUR UTAMA
### Customer Side
- Login/Register customer  
- Lihat menu per kategori  
- Status menu **tersedia / habis** berdasarkan stok  
- Add to cart, update qty, remove cart  
- Checkout (pilih meja + catatan opsional)
- Reservation (pilih tanggal, jam, meja + catatan opsional) 
- About Us & Contact  

### Admin Side (Admin & Kasir)
- Login staff (admin/kasir)  
- Kelola menu + upload foto  
- Kelola stok menu  
- Kelola meja, kategori, akun staff & customer
- Kelola reservasi
- Pembayaran pesanan + cetak receipt
- Pembayaran pesanan + cetak receipt  
- Report penjualan, transaksi, pendapatan, dan menu terlaris  

## 🛠️ Cara Menjalankan
1. Jalankan **XAMPP (Apache + MySQL)**
2. Import database ke phpMyAdmin
3. Atur koneksi database di `config.php`
4. Jalankan: http://localhost/restoran/
